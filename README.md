# fortivo
Keep your sensitive information in one place, secured
## About
Ever thought "Damn I don't know where the fuck to save my passwords"?\
Well thats one use case for fortivo, it produces files called arca(s), these arca(s) store your passwords in objects called arcanum(s), these arca(s) are encrypted by default
## Submodules
fortivo is divided in two submodules:
- website -> ``fortivo-website/``
- engine -> ``fortivo-engine/``
## Current state of the project
fortivo is not ready yet, it should not be used until it hits v1.0.0, which I foresee it will be around the end of 2025
